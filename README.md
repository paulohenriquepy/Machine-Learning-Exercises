# Machine Learning Exercises

Este repositorio reúne una colección de notebooks en Python diseñados para practicar y aplicar conceptos de Machine Learning, utilizando tanto datos simulados como datos reales. El objetivo principal es consolidar habilidades de análisis, modelización y evaluación de modelos predictivos.

# Estructura de los notebooks

## Predizer_Abundancia_Bacterias.ipynb
Ejercicio de regresión o clasificación para estimar la abundancia de bacterias a partir de variables ambientales.

## Predizer_Precos_Carros.ipynb
Modelización orientada a la predicción de precios de automóviles empleando datos tabulares y técnicas de regresión supervisada.

## Predizer_Vendas.ipynb
Análisis de series históricas de ventas y aplicación de modelos para prever la demanda futura.

Tecnologías utilizadas

Python 3.x

Scikit-learn

Pandas

NumPy

Matplotlib / Seaborn
